A beautiful template for React Native, Support scaling size.
## Usage
```js
npx react-native init MyApp --template react-native-template-components
```

#### React 18.0.0
#### React Native 0.69.4
#### React Navigation V6
#### React Native Vector Icons
#### Redux, Redux Observable
#### I18Next
